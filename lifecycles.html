React State & Lifecyles

State and lifecycles are important concepts in the React library.

This page will breifly explain the concepts and have an interactive demo along with a code snipet for them.

In React, we have components which can be anything like a textbox, button or a graph visual. Components can be stateful by having a JavaScript object called state. This state object stores dynamic data and determines the behavior of the component. State is private to the component and what that means is that it can only be controlled by the component itself. Another important detail about state is that it is immutable.

            class Greeting extends React.Component {
