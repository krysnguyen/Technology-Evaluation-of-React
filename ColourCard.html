Color Card Components

This page is a simple demonstration of how React Components can be used.
React components are reusable chunks of JavaScript that output (via JSX) HTML elements.

Building apps with a very complex UI requires a simple solution to overcoming these challenges. React tries to solve this by making an app's visual pieces self-contained and reuseable.
In the example below are simple colour palette cards, you can find these cards at a home improvement store if you wanted to pick out a paint colour. To recreate these cards with components we need to determine:

  1. Major visual elements
  2. What each component will look like
After breaking down the cards, they are now composed of 2 componets. The colour square and the white rectangle label with the RGB colour inside of it.
Both of these components will live inside their parent component. The component hierarchy will look like this:
  • (Parent) Card
    • (Children) Colour square
    • (Children) Label
So we will have 3 components total. Because of this hierarchy we can pass the colour to the Label and to the background colour of the square as seen in the code below.
To generalize the result of components, each components render() function returns some HTML to another components render function. The HTML accumulates from each components render call until all the HTML is pushed to the parent component and it is displayed to the DOM.
